Wooting/XInput Passthrough/Mouse Steering/Digital to XInput. Quick and easy profile switching through Alt+Y or by holding Ctrl+Shift on script start.
Requires installing ViGEmBus, and optionally Interception driver for games that need raw input suppression for mouse rebinds that can't be done through Autohotkey alone. Do mind the Interception device add limit bug.
Contains modified version of ViGEmBusWrapper script with fixed analog translations and full analog ranges, and new lightweight Wooting sdk wrapper.
